# Explainability Module

Generates human-readable explanations for each signal, describing the key factors that drove the decision.

## Planned Components

- `explanation_builder.py` — Constructs natural language signal rationale
- `factor_renderer.py` — Formats individual contributing factors for display
- `report_generator.py` — Builds full signal reports with charts and annotations
