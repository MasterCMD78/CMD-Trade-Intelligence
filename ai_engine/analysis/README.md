# Analysis Engine

This module will contain market analysis logic including technical pattern detection, trend analysis, and multi-timeframe confluence checks.

## Planned Components

- `pattern_detector.py` — Candlestick and chart pattern recognition
- `trend_analyzer.py` — Trend direction and strength analysis
- `confluence_checker.py` — Multi-indicator confluence scoring
- `timeframe_analyzer.py` — Cross-timeframe signal alignment
