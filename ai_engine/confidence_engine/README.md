# Confidence Engine

Produces confidence scores for each generated signal using a weighted multi-factor model.

## Planned Components

- `confidence_scorer.py` — Aggregates factor weights into a 0–100 confidence score
- `factor_weights.py` — Configurable weight definitions per factor
- `historical_validator.py` — Back-tests confidence model against historical signals
