# Market Data Providers

Adapters for connecting to external market data sources (REST and WebSocket).

## Planned Providers

- `forex_provider.py` — Forex tick and OHLCV data (FXCM, OANDA, or broker API)
- `crypto_provider.py` — Crypto market data (Binance, Coinbase, Kraken)
- `provider_interface.py` — Abstract base class all providers must implement
