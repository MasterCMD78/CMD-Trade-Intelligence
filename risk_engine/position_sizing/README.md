# Position Sizing Engine

Computes recommended position sizes based on account balance, risk tolerance, and current volatility.

## Planned Components

- `position_sizer.py` — Fixed fractional and ATR-based position sizing
- `lot_calculator.py` — Forex-specific lot size computation (standard, mini, micro)
- `exposure_checker.py` — Checks total portfolio exposure limits before trade entry
