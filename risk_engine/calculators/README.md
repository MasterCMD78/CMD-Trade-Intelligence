# Risk Calculators

Core risk calculation utilities used across position sizing and portfolio-level risk management.

## Planned Components

- `drawdown_calculator.py` — Max drawdown, current drawdown, and recovery metrics
- `risk_reward_calculator.py` — R:R ratio computation from entry/SL/TP
- `volatility_calculator.py` — ATR-based volatility measurements
- `kelly_criterion.py` — Kelly formula for optimal position sizing (optional)
